## VoiceRecorder

VoiceRecorder is a simple voice/audio recorder for my wife and for fun.

The program is written in Python 3.6 and PyQt5. [pydub](https://github.com/jiaaro/pydub) is used to encode audio files into OGG/Vorbis.
